# Vibol Khan Portfolio Website

A clean responsive portfolio website built with HTML, CSS, and JavaScript only.

## Project Structure

```text
vibol-khan-portfolio/
├── index.html
├── style.css
├── script.js
└── README.md
```

## How to Run

1. Extract the ZIP file.
2. Open `index.html` in your browser.
3. Edit the content in `index.html` to update your profile, projects, or contact details.

## Sections Included

- Hero section
- About section
- Skills section
- Projects section
- Experience timeline
- Education section
- Contact section

## Built With

- HTML5
- CSS3
- Vanilla JavaScript
